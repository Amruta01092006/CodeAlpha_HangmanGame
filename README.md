# CodeAlpha_HangmanGame

## Description

This project is a text-based implementation of the classic Hangman game. The program randomly selects a word, and the player has to guess it one letter at a time. The game continues until the player either guesses the word correctly or runs out of attempts.

## Features

- Random word selection
- Tracks guessed letters
- Limited number of attempts
- Input validation
- Win and lose conditions
- Simple terminal-based interface

## Technologies Used

- Python 3
- Random module (built-in)

## How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python CodeAlpha_HangmanGame.py
```
